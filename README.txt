Инструкция для установки Telegram-бота
1. Установите Python
Скачайте последнюю версию Python с официального сайта: https://www.python.org/downloads/.
Во время установки обязательно отметьте опцию Add Python to PATH, чтобы Python добавился в системные переменные
2. Скачайте файлы
Скачайте файлы и сохраните их в любую папку на вашем компьютере.
3.Установите зависимости
Откройте терминал или командную строку (нажмите Win+R, введите cmd, затем нажмите Enter).
Перейдите в папку, где находится файл bot.py. Например:
cd путь_к_папке
Установите необходимые библиотеки (предварительно, установите PIP на ваш компьютер):
pip install python-telegram-bot
4.Получите ваш ID пользователя в телеграме:
Запустите бота в Telegram: @userinfobot.
Отправьте команду /start.
Бот ответит информацией о вашем аккаунте, включая ваш chat_id.
5. В файл .env, добавьте ваш ID в строку ORGANIZER_CHAT_ID МЕЖДУ КОВЫЧЕК
6. Запустите бота
В том же терминале выполните команду:
python bot.py
Если все настроено правильно, бот запустится и будет готов к работе.
7. Использование
Откройте Telegram и найдите вашего бота по имени.
Отправьте боту команду /start.


ПРИМЕЧАНИЕ:

Для перезапуска бота просто повторите команду python bot.py

Возможно, для работы бота понадобится установить еще библиотеку python-dotenv
Для этого, в терминале выполните команду: pip install python-dotenv

P.S. Менять .env файл желательно в Visual Studio Code.
